# Kernel helper functions

A collection of helpers to generate kernel matrices

## Usage

``` r
kernel_square(size = 3)

kernel_circle(size = 3)

kernel_gaussian(size = 3, sigma = 1)

kernel_plot(x)
```

## Arguments

- size:

  integer size of the kernel matrix to generate. Must be an odd number
  so that the central pixel is actually central.

- sigma:

  numeric the sigma parameter for the gaussian kernel

- x:

  a kernel to visualize

## Value

a kernel matrix (or a plot for kernel_plot)
